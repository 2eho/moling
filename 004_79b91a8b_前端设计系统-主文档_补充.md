## §17 后台管理页

> **文件路径**：`moling-web/src/app/admin/page.tsx`
> **路由**：`/admin`
> **状态**：✅ 已实现

### 页面概述

后台管理页面，包含 5 个 Tab：
1. **系统概览** - 统计卡片 + 图表 + 最近活动
2. **用户管理** - 用户列表 + 筛选 + 分页
3. **LLM 用量** - LLM 调用统计 + 趋势图
4. **支付记录** - 支付订单列表 + 筛选
5. **LLM 配置** - LLM API Key 配置（已实现）

### LLM 配置 Tab（已实现）

**功能**：
- 输入 API Key（password 类型，脱敏显示）
- 输入 API Base URL
- 选择默认模型
- 测试连接
- 保存配置

**API 对接**：
- `GET /api/v1/admin/llm-config` - 获取当前配置
- `POST /api/v1/admin/llm-config` - 保存配置
- `POST /api/v1/admin/llm-config/test` - 测试连接

**交互事件**：
| 触发器 | 动作 |
|--------|------|
| 页面加载 | `GET /api/v1/admin/llm-config` → 填充表单 |
| "测试连接" 点击 | `POST /api/v1/admin/llm-config/test` → 显示结果 |
| "保存配置" 点击 | `POST /api/v1/admin/llm-config` → 成功 Toast |
| API Key 输入 | 实时检查格式（以 `sk-` 开头）|

## §18 编辑项目页

> **文件路径**：`moling-web/src/app/projects/[projectId]/edit/page.tsx`
> **路由**：`/projects/[id]/edit`
> **状态**：✅ 已实现

### 页面概述

编辑项目信息页面，只包含项目名称编辑。

### 布局结构

```
┌────────────────────────────────┐
│  Card                          │
│  ┌──────────────────────┐      │
│  │ 标题：编辑项目       │      │
│  └──────────────────────┘      │
│  ┌──────────────────────┐      │
│  │ 标签：项目名称       │      │
│  │ [输入项目名称    ]   │      │
│  └──────────────────────┘      │
│  ┌──────────────────────┐      │
│  │ [取消] [保存]       │      │
│  └──────────────────────┘      │
└────────────────────────────────┘
```

### 数据模型

```typescript
interface EditProjectPageData {
  projectId: string;
  title: string;
  loading: boolean;
  saving: boolean;
}
```

### 交互事件

| 触发器 | 动作 |
|--------|------|
| 页面加载 | `GET /api/v1/projects/:id` → 填充表单 |
| "保存" 点击 | `PUT /api/v1/projects/:id` → 成功 → 跳转 `/workspace/:id` |
| "取消" 点击 | `router.back()` |

### API 对接

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/v1/projects/:id` | GET | 获取项目详情 |
| `/api/v1/projects/:id` | PUT | 更新项目 |
