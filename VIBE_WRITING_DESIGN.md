# Vibe Writing — 革命性网文写作产品设计文档

## 一、设计哲学：从 Codex 到 Vibe Writing

### Codex 对编程行业的颠覆性创新

| Codex 原则 | 编程领域实现 | Vibe Writing 映射 |
|-----------|-------------|------------------|
| Agent of Agents | 多专业 Agent 协作编码 | 剧情/人物/对话/风格/世界观 Agent 协作写作 |
| Intent > Manual Labor | 描述意图，Agent 生成代码 | 选择方向，Agent 生成内容 |
| Heavy Options, Light Input | 预设选项推进，减少手动输入 | A/B/C 选项 + 最小化 D 自由输入 |
| Context is King | 深度理解项目上下文 | 深度理解小说状态、阶段、风格 |
| Flow State | 不打乱开发者心流 | 不打乱写作者创作流 |
| Parallel Execution | 并行多任务处理 | 并行多维度写作推进 |
| Spec-Driven | 先规格后实现 | 先构思后写作 |

### Vibe Writing 核心原则

1. **选择优于输入 (Choose > Type)** — 80% 操作是选 A/B/C，20% 是必要自由输入
2. **代理协作 (Agent Collaboration)** — 多个 AI Agent 各司其职，协调产出
3. **上下文自适应 (Context-Adaptive)** — 系统深度理解当前小说阶段，智能生成选项
4. **心流保护 (Flow Protection)** — 绝不打断创作者思路，无缝衔接上下文
5. **渐进式展开 (Progressive Disclosure)** — 从宏观到微观，逐步细化

```
[构思] → [大纲] → [人设] → [世界构建] → [草稿] → [修订]
  │        │        │         │           │        │

核心概念  章节结构  角色卡片   世界观设定   逐章写作   审校润色
类型定位  故事节拍  人物弧光   规则体系    AI 生成    一致性检查
预设卖点  冲突布局  关系网络   环境氛围    选项推进   文风统一
```

### 交互流

```
1. 用户进入写作阶段
2. Coordinator 分析上下文，调用相关 Agent
3. 各 Agent 基于小说状态生成提案
4. Coordinator 整合提炼为 A/B/C 选项 + 预览
5. 用户选择或输入
6. 所选 Agent 执行，更新小说状态
7. 循环 → 回到步骤 2
```

### 数据模型

```python
class Novel(BaseModel):
    id: str
    title: str
    genre: str  # 玄幻/都市/言情/...
    phase: Phase  # 构思/大纲/人设/世界/草稿/修订
    current_chapter: int
    style_profile: StyleProfile
    world_rules: dict

class AgentProposal(BaseModel):
    agent: str  # plot/character/dialogue/style/world
    option_label: str  # A/B/C/D
    title: str
    description: str
    preview: str  # 生成内容预览
    confidence: float
    impact_analysis: dict

class WritingContext(BaseModel):
    novel_summary: str
    previous_chapter: str
    current_scene: str
    active_characters: list
    pending_foreshadowing: list
    style_guidelines: dict
```

## 六、产品差异化对比

| 维度 | 传统写作工具 | AI 写作助手 | Vibe Writing |
|------|------------|-----------|-------------|
| 交互方式 | 全手动输入 | 对话式交互 | 选择式推进 |
| AI 参与度 | 无 | 单模型对话 | 多代理协作 |
| 上下文理解 | 无 | 当前对话窗口 | 全小说深度理解 |
| 创作心流 | 频繁中断 | 对话中断 | 无缝推进 |
| 学习曲线 | 低 | 中 | 极低 |
| 专业深度 | 取决于作者 | 通用 | 网文专精 |
| 质量一致性 | 不稳定 | 不稳定 | 多代理校验 |
