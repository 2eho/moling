## 变更 1：LLM 配置页面

### 页面功能

后台管理新增 **"LLM 配置"** Tab，允许管理员动态修改 LLM 配置，无需重启服务。

### 页面布局

```
┌────────────────────────────────────────┐
│  后台管理                              │
├────────────────────────────────────────┤
│  [概览] [用户] [LLM 用量] [LLM 配置] [支付]  │  ← Tab Bar
├────────────────────────────────────────┤
│  ┌──────────────────────────────────┐ │
│  │  LLM 配置                            │ │
│  │                                    │ │
│  │  API Base:                         │ │
│  │  [https://api.deepseek.com    ]   │ │
│  │                                    │ │
│  │  API Key:                          │ │
│  │  [sk-xxxx                        ]   │ │
│  │  ✅ 已验证（green checkmark）       │ │
│  │                                    │ │
│  │  模型:                              │ │
│  │  [deepseek-chat              ▼]   │ │
│  │                                    │ │
│  │  [测试连接]  [保存配置]            │ │
│  └──────────────────────────────────┘ │
└────────────────────────────────────────┘
```

### 数据模型

```typescript
interface LLMConfigData {
  apiBase: string;
  apiKey: string;        // 输入时明文，保存后脱敏显示
  model: string;
  isConfigured: boolean;
  apiKeyMasked: string;  // "sk-ab12****wxyz"
}

interface LLMConfigTestResult {
  success: boolean;
  message: string;       // "连接成功" | "连接失败：..."
  responseTime?: number;  // 响应时间（ms）
}
```

### 交互事件

| 触发器 | 动作 |
|--------|------|
| 页面加载 | GET `/api/v1/admin/llm-config` → 填充表单（API Key 脱敏） |
| "测试连接" 点击 | POST `/api/v1/admin/llm-config/test` → 显示结果 Toast |
| "保存配置" 点击 | POST `/api/v1/admin/llm-config` → 成功 Toast → 更新表单（API Key 脱敏） |
| API Key 输入 | 实时检查格式（以 `sk-` 开头）→ 显示 ✅/❌ |
| 保存成功 | 显示 "配置已保存，将在下次 LLM 调用时生效" |
