## 常见问题

### Q1: 后端启动失败（greenlet错误）
**A:** Windows上可能会有greenlet DLL问题。解决方法：
1. 使用SQLite数据库（已在.env中配置）
2. 暂时禁用Redis（Redis连接失败会优雅降级）
3. 检查`.venv`虚拟环境是否正确激活

### Q2: 前端无法调用后端API
**A:** 检查CORS配置：
1. 后端`.env`中的`CORS_ORIGINS`包含前端URL
2. 前端`apiClient.ts`中的baseURL指向正确后端地址

### Q3: 数据库表未创建
**A:** 应用启动时会自动创建表（SQLite）。如果失败：
```bash
cd C:\Users\Admin\Desktop\MolingProject\moling-server
python -c "from app.models.base import Base; from sqlalchemy import create_engine; Base.metadata.create_all(create_engine('sqlite:///moling.db'))"
```
