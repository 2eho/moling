# 墨灵 UI 专业化改造报告

**日期**: 2026-06-16  
**版本**: v0.3.0  
**改造范围**: Web端侧边栏 + 移动端底部导航 + 专注模式 + 手势操作 + 工作台布局优化

---

## 执行摘要

本次改造参考 **Obsidian + Notion** 设计理念，实现"随时随地可访问，但不打扰写作"的专业写作平台体验。

### 改造前后对比

| 维度 | 改造前 | 改造后 |
|------|:--------:|--------|
| **Web端导航** | 顶部导航栏（占用编辑空间） | 可折叠侧边栏（56px/280px） |
| **移动端导航** | 5个底部导航项（拥挤） | 3个主导航 + 更多菜单 |
| **专注模式** | ❌ 无 | ✅ Esc键 + `?focus=true` URL参数 |
| **手势操作** | ❌ 无 | ✅ 左滑/右滑/上滑/下滑 |
| **工作台布局** | 三栏固定（左/中/右） | Obsidian风格（可折叠 + 标签式） |
| **移动端面板** | 堆叠显示（拥挤） | 抽屉式（左/右滑出） |
| **页面动画** | ❌ 无 | ✅ 6种CSS动画（缓动） |

## 1. Web端侧边栏（参考 Obsidian）

### 功能实现

**文件**: `src/components/layout/Sidebar.tsx` + `Sidebar.module.css`

| 功能 | 说明 |
|------|------|
| **可折叠** | 56px（只显示图标）/ 280px（完整），localStorage持久化 |
| **项目切换器** | 下拉菜单显示最近项目，点击切换到 `/workspace/[id]` |
| **导航菜单** | 按使用频率排序：项目 → 写作 → 四库 → 通知 → 设置 |
| **活跃状态** | `usePathname()` 自动高亮当前页面 |
| **平滑动画** | 所有过渡使用 `var(--ease-spring)` 缓动 |

### 导航菜单设计

```
✒️ 墨灵 (Logo，点击回到 /projects)
━━━━━━━━━━━━━━
📚 项目          ← /projects
✒️ 写作          ← /workspace/[lastProjectId]
📖 四库          ← /vaults/[lastProjectId]
🔔 通知          ← /notifications
━━━━━━━━━━━━━━
⚙️ 设置          ← /settings (底部)
```

**设计理念**:
- 写作是核心功能，放在第二位（第一位是项目管理）
- 设置放在底部（最不常用）
- 所有导航项都有图标 + 文字（可折叠到只显示图标）

## 2. 移动端底部导航（3项 + 更多菜单）

### 功能实现

**文件**: `src/components/layout/BottomNav.tsx` + `BottomNav.module.css`

| 功能 | 说明 |
|------|------|
| **主导航（3项）** | `[ ✒️ 写作 ] [ 📖 四库 ] [ ⋮ 更多 ]` |
| **更多菜单** | 从底部弹出：切换项目 / 通知 / 设置 / 个人资料 |
| **自动隐藏** | 编辑器全屏时，上滑显示，下滑隐藏 |
| **安全区适配** | `env(safe-area-inset-bottom)` 适配 iPhone X+ |
| **弹出动画** | `menuSlideUp` 动画（0.3s，`var(--ease-spring)`） |

### 手势操作（预留接口）

**文件**: `src/hooks/useTouchGesture.ts`

| 手势 | 动作 | 说明 |
|------|------|------|
| **左滑** | 打开参考面板（人物/时间线/剧情/世界观） | 分发 `open-reference-panel` 事件 |
| **右滑** | 打开 AI 面板（抽卡/生成/建议） | 分发 `open-ai-panel` 事件 |
| **上滑** | 显示底部导航 | 退出专注模式 |
| **下滑** | 隐藏底部导航 | 进入专注模式 |
| **点击** | 非滑动操作 | 正常点击处理 |

**技术实现**:
- 使用 `TouchEvent` 监听（`touchstart` + `touchend`）
- 可配置滑动阈值（默认50px）
- 支持指定目标元素（默认 `document`）
- 被动事件监听（性能优化）

## 3. 专注模式（Esc键 + URL参数）

### 功能实现

**文件**: `src/components/layout/AppShell.tsx`

| 功能 | 说明 |
|------|------|
| **Esc键切换** | 按 `Esc` 键进入/退出专注模式 |
| **URL参数** | `?focus=true` 激活专注模式（可分享链接） |
| **自动隐藏导航** | 专注模式下，Navbar/BottomNav/Sidebar 全部隐藏 |
| **全屏编辑器** | 编辑器占满整个屏幕（`position: fixed; z-index: 9999`） |

### 使用场景

| 场景 | 操作 | 效果 |
|------|------|------|
| **深度写作** | 按 `Esc` 或添加 `?focus=true` | 隐藏所有导航，全屏编辑 |
| **参考查看** | 左滑（移动端） | 打开参考面板（不退出专注模式） |
| **AI辅助** | 右滑（移动端） | 打开 AI 面板（不退出专注模式） |
| **退出专注** | 按 `Esc` 或上滑（移动端） | 显示所有导航 |

## 4. 工作台布局优化（参考 Obsidian）

### 功能实现

**文件**: `src/app/workspace/[projectId]/page.tsx` + `workspace.module.css`

| 功能 | 改造前 | 改造后 |
|------|--------|--------|
| **左面板** | 固定250px，列表式 | 可折叠 + 标签式切换（人物/时间线/剧情/世界观） |
| **编辑器** | 全宽 | 居中，最大宽度900px（Obsidian风格） |
| **右面板** | 固定320px，列表式 | 可折叠 + 标签式切换（AI建议/抽卡/生成历史） |
| **移动端** | 堆叠显示 | 抽屉式（左/右滑出 + 遮罩层） |

### Obsidian风格设计

```
┌────────┬──────────────┬──────────┐
│ 左面板  │  编辑器        │  右面板    │
│ 可折叠  │  居中最大宽    │ 可折叠    │
│ 标签式  │  900px        │ 标签式    │
└────────┴──────────────┴──────────┘
```

**关键改进**:
1. **编辑器居中** - 最大宽度900px（阅读舒适度最佳）
2. **标签式切换** - 左/右面板使用标签切换（不滚动）
3. **可折叠** - 点击按钮隐藏面板（专注写作）
4. **移动端抽屉** - 左/右面板改为抽屉式（不拥挤）

## 5. 页面切换动画

### 功能实现

**文件**: `src/app/globals.css`

| 动画类 | 效果 | 使用场景 |
|---------|------|----------|
| `.pageEnter` | 淡入 + 上移10px | 所有页面进入 |
| `.pageFadeIn` | 淡入 | 模态框/下拉菜单 |
| `.pageSlideInLeft` | 左滑入 | 左面板打开 |
| `.pageSlideInRight` | 右滑入 | 右面板打开 |
| `.scaleIn` | 缩放进入 | 卡牌抽取 |
| `.cardDraw` | 卡牌抽取动画 | 抽卡结果展示 |
| `.pulse` | 脉冲动画 | 加载指示器 |
| `.spin` | 旋转动画 | 加载指示器 |

**技术实现**:
- 所有动画使用 `var(--ease-spring)` 或 `var(--ease-default)` 缓动
- 动画持续时间：0.3s（标准） / 0.5s（复杂动画）
- 支持 `prefers-reduced-motion` 媒体查询（无障碍）

## 6. 技术实现细节

### 6.1 状态管理

| 状态 | 存储位置 | 持久化 |
|------|----------|----------|
| **侧边栏折叠** | `AppShell.tsx` (`useState`) | ✅ localStorage (`moling-sidebar-collapsed`) |
| **专注模式** | `AppShell.tsx` (`useState`) | ✅ URL参数 (`?focus=true`) |
| **移动端检测** | `AppShell.tsx` (`useState`) | ❌ 运行时检测（`window.innerWidth`） |
| **手势状态** | `useTouchGesture.ts` (`useState`) | ❌ 运行时状态 |

### 6.2 事件通信

| 事件名 | 分发者 | 监听者 | 说明 |
|---------|--------|--------|------|
| `open-reference-panel` | `AppShell.tsx` (手势) | `workspace/[projectId]/page.tsx` | 打开参考面板 |
| `open-ai-panel` | `AppShell.tsx` (手势) | `workspace/[projectId]/page.tsx` | 打开 AI 面板 |

**技术实现**:
- 使用 `CustomEvent` 分发事件
- 使用 `window.addEventListener` 监听事件
- 事件 payload：无（只是通知）

### 6.3 响应式断点

| 断点 | 设备 | 布局 |
|--------|------|------|
| **> 768px** | Web端/平板横屏 | 侧边栏 + 主内容区 |
| **≤ 768px** | 移动端/平板竖屏 | 顶部导航 + 底部导航 + 抽屉面板 |

**技术实现**:
- 使用 `useEffect` + `window.addEventListener("resize")` 检测
- 使用 CSS `@media (max-width: 768px)` 响应式样式
- 使用 `env(safe-area-inset-bottom)` 适配 iPhone X+ 安全区
