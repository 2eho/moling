## 五、使用说明

### 数据库迁移

模型变更后，需要创建数据库迁移脚本：

```bash
cd moling-server
alembic revision --autogenerate -m "修复数据模型 - UUID 主键和字段对齐"
alembic upgrade head
```

### 环境变量

确保以下环境变量已配置：

```
# 数据库
DATABASE_URL=postgresql+asyncpg://user:password@localhost:5432/moling

# Redis
REDIS_URL=redis://localhost:6379/0

# DeepSeek API
DEEPSEEK_API_KEY=your_api_key
DEEPSEEK_API_BASE=https://api.deepseek.com/v1
```
