### 修复内容

#### 修复 1：实现四库过滤算法（TODO 1）
**位置**：`_build_layer2()` 方法（第 175-338 行）

**修改前**：
- 读取所有四库数据（人物、剧情承诺、世界观规则）
- 没有根据卡片关联进行过滤

**修改后**：
1. 从数据库读取所选卡片
2. 提取卡片关联的所有 ID（人物 ID、剧情承诺 ID、世界观规则 ID）
3. 只读取这些 ID 对应的四库条目
4. 如果没有关联 ID，则读取前 5 个条目作为参考

**代码变更**：
```python
# 1. 从数据库读取卡片
stmt = select(CardPool).where(
    CardPool.id.in_(card_ids),
    CardPool.project_id == project_id
)
result = await db.execute(stmt)
cards = result.scalars().all()

# 2. 提取卡片引用的所有 ID
character_ids = []
promise_ids = []
world_rule_ids = []

for card in cards:
    # 提取人物 ID
    if card.characters:
        for char in card.characters:
            if isinstance(char, dict) and 'id' in char:
                character_ids.append(char['id'])
            elif isinstance(char, str):
                character_ids.append(char)
    # ... 类似处理剧情承诺和世界观规则

# 去重
character_ids = list(set(character_ids))
promise_ids = list(set(promise_ids))
world_rule_ids = list(set(world_rule_ids))

# 3. 只读取关联的条目
if character_ids:
    stmt = select(VaultCharacter).where(
        VaultCharacter.id.in_(character_ids),
        VaultCharacter.project_id == project_id
    ).limit(5)
    # ...
```

#### 修复 2：调用 fingerprint_to_prompt()（TODO 2）
**位置**：`_build_layer4()` 方法（第 389-422 行）

**修改前**：
```python
if project and project.style_fingerprint:
    # TODO: 调用 style_prompt_builder.fingerprint_to_prompt()
    # 当前简化：直接注入 style_fingerprint
```

**修改后**：
```python
if project and project.style_fingerprint:
    # 调用 style_prompt_builder.fingerprint_to_prompt()
    try:
        style_constraint = fingerprint_to_prompt(project.style_fingerprint)
        layer4_parts.append(style_constraint)
    except Exception as e:
        # 如果转换失败，直接注入原始指纹
        layer4_parts.append("【风格指纹】")
        layer4_parts.append(str(project.style_fingerprint))
```

**导入添加**：
在第 22 行添加了导入语句：

