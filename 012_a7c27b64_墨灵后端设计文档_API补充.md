## Generation API

### POST /api/v1/generation

创建生文任务（异步）。

**请求体**：

```json
{
  "project_id": "uuid",
  "chapter_id": "uuid",
  "card_ids": ["uuid1", "uuid2", "uuid3"],
  "weave_pattern_id": "uuid",
  "weights": {
    "card_1": 0.8,
    "card_2": 0.6,
    "card_3": 1.0
  }
}
```

**响应示例**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "task_id": "uuid",
    "status": "pending",
    "estimated_seconds": 15
  }
}
```

**实现逻辑**：

1. 校验用户权限（项目归属）
2. 校验卡牌状态（新鲜期、未退役）
3. 创建 Celery 任务（`generation_task.apply_async()`）
4. 返回 `task_id`

**状态码**：

- `201`：任务创建成功
- `400`：参数错误
- `403`：权限不足
- `409`：该章节正在生文中（冲突）

---

### GET /api/v1/generation/:task_id

查询生文任务状态。

**响应示例**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "task_id": "uuid",
    "status": "processing",
    "progress": 60,
    "result": null,
    "error": null
  }
}
```

**状态枚举**：

- `pending`：待处理
- `processing`：处理中
- `completed`：已完成
- `failed`：失败

---

### DELETE /api/v1/generation/:task_id

取消生文任务。

**状态码**：

- `204`：取消成功
- `400`：任务已完成或失败（无法取消）

## Notification API

### GET /api/v1/notifications

获取通知列表。

**查询参数**：

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| `unread_only` | boolean | 否 | 只返回未读通知 |
| `limit` | integer | 否 | 返回数量限制（默认 20） |
| `offset` | integer | 否 | 分页偏移（默认 0） |

**响应示例**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "notifications": [
      {
        "id": "uuid",
        "type": "phase4_completed",
        "title": "收纳完成",
        "message": "第 5 章的收纳已完成",
        "is_read": false,
        "created_at": "2026-06-14T10:00:00Z"
      }
    ],
    "total": 15,
    "unread_count": 3
  }
}
```

---

### GET /api/v1/notifications/unread-count

获取未读通知计数。

**响应示例**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "unread_count": 3
  }
}
```

---

### POST /api/v1/notifications/:notification_id/read

**状态码**：`200`

---

### POST /api/v1/notifications/read-all

**状态码**：`200`

---

### DELETE /api/v1/notifications/:notification_id

删除通知。

**状态码**：`204`

## Card API 补充

### POST /api/v1/cards/:card_id/retire

手动退役卡牌。

**状态码**：`204`

**实现逻辑**：

1. 检查卡牌是否存在且属于该项目
2. 设置 `status = "retired"`
3. 记录 `retired_at = now()`

---

### GET /api/v1/cards/history

获取抽卡历史。

**查询参数**：

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| `chapter_id` | string | 否 | 过滤特定章节 |
| `limit` | integer | 否 | 返回数量限制（默认 50） |

**响应示例**：

```json
{
  "code": 0,
  "message": "ok",
  "data": [
    {
      "id": "uuid",
      "chapter_id": "uuid",
      "card_id": "uuid",
      "card_title": "神秘人身份揭露",
      "drawn_at": "2026-06-14T10:00:00Z",
      "is_used": true

}
```
